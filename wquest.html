The Wage Quest

Created as an entry in the first anual WIC Hackathon in Suny Oswego, this text-based game was created using the Java Swing library. It is a game that touches on inequality in the workplace for women. It's built on a system of try-catch blocks that enable the user to make choices that lead to different results.
